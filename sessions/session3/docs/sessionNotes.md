# Session 3 Notes  - a survey of private cloud infrastructure

This is an incomplete list of the most important open source cloud infrastructure projects

## OpenStack

https://www.openstack.org/ OpenStack was launched in 2010 and its primary objective was to resemble the behaviour of leading public clouds. This is why AWS, Azure, or Google Cloud workloads can be moved to OpenStack so easily.

Open Stack is widely used in tee Telecoms industry for 

* ETSI NFV (network function virtualisation) https://osm.etsi.org/news-events/blog/57-pairing-openstack-and-open-source-mano-for-nfv-deployments

and OpenRAN 

* https://www.telecominfraproject.com/openran   Open Radio Access Network (Open RAN) 

* https://canonical.com/solutions/telco/open-ran

Various Open Stack distributions

* Open Stack on Ubuntu https://ubuntu.com/openstack

* RDO -  OpenStack on RHEL/CENTOS RDO is the RPM Distribution of OpenStack for RHEL derivative https://www.rdoproject.org/deploy/
* KOLLA - Open Stack Docker Containers  https://docs.openstack.org/kolla-ansible/latest  OpenStack Kolla is an official OpenStack project that provides Docker containers and Ansible playbooks to deploy OpenStack services quickly, securely, and with production-ready standards.
* open stack on VMWare https://docs.openstack.org/nova/pike/admin/configuration/hypervisor-vmware.html
